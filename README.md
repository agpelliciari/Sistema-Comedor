# Sistema para Comedor Universitario

En este repositorio se encuentra la documentacion para nuestra aplicacion web del sistema para comedores universitarios. 

El codigo fuente de la aplicacion se encuentra en el [repositorio](https://gitlab.com/ayudantes-ingsoft/sistema-comedor-grupo-09) privado.

## Que ofrece nuestra aplicacion? 

Nuestro sistema de gestion para comedores universitarios busca modernizar la experiencia de
estudiantes y personal, permitiendo a los estudiantes consultar el menu, armar, realizar y pagar
pedidos desde una interfaz web, y brindando al personal herramientas para gestionar pedidos,
preparacion en cocina y disponibilidad de productos en tiempo real. Adicionalmente, facilita la
administracion del menu, stock, promociones y accesos del personal.
